# 🌳 Task 8: Tumor Classification using Random Forest Ensembles

📘 **Data Science Internship - EduTech Solutions**

---

### 🎯 Objective
To master ensemble learning methods by building and contrasting a Random Forest Classifier against a standalone Decision Tree to evaluate variance reduction techniques.

* **Core Outputs:** `accuracy_comparison_report.txt` and `feature_importance_plot.png` plotting feature impacts.
